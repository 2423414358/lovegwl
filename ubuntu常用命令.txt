													ubuntu 常用命令
一个完整的指令的标准格式：linux通用格式											
	#指令主体[选项[操作对象]
	一个指令可以包括多个选项
	操作对象也可以是多个
	
	
	
基础指令:
	ls	pwd	cd	mkdir	touch	rm	vim	输出重定向	cat

ls指令:	含义：	列出
	用法一：
		ls 含义: 列出当前工作目录下的所有文件/文件夹的名称
		
	用法二:
		ls 路劲	含义列出指定路径下的所有文件/文件夹的名称
	用法三:
		#ls 选项 路径	含义：在列出指定路径下的文件/文件夹的名称 并以指定的格式进行显示
			常见的语法：
				ls -l 路径	表示以详细列表的形式进行展示
				ls -la 路径	表示所有的文件/文件夹(包含了隐藏文件/文件夹)
				ls -lh 路径	列出指定路径下的所有文件/文件夹的名称 一列表的形式并且在显示文档大小的时候以较高的形式显示
				
				
	ls 列出的结果颜色说明 其中蓝色的名称表示文件夹 白色的表示文件 绿色的其权限拥有所有权限
				
pwd指令:
	用法:pwd	含义	(print working directory)打印当前工作目录
			
												

cd 指令:	(change directory, 改变目录)

	cd ..返回上一级 	cd . 切换用户目录	cd ~	返回home目录	cd - 返回上一个目录		
	


mkdir指令:
	mkdir 路径 文件夹名
	mkdir -p 路径 文件夹名	当一次性创建多层不存在的目录的时候 添加-p参数 否则会报错
	mkdir 	路径1 路径2 路径3 ... 	表示一次性创建多个目录
	
	可以指定路径也可以不指定路径 不指定文件则表示创建在当前路径下
	
touch指令:	创建文件
	touch 文件路径 文件名称
	touch 文件路径 文件名称1 文件名称2 
	
	可以指定路径也可以不指定路径 不指定文件则表示创建在当前路径下
	
	
cp 命令:	复制文件/文件夹到指定的位置
	cp 被复制的文件路径 文档被复制到的路径
	cp -r 被复制的文件夹路径 文件夹被复制到的路径
	当使用cp指令来复制文件夹的时候需要添加 -r 【-r表示递归】
	

mv 指令	(move, 移动，剪切)	
	mv 需要移动的文档路径 需要移动的位置
	
	
rm指令	remove移除、删除
	rm 选项 需要移除的文档路径||名称
	选项:
		-f 	不提示删除选项
		-r	删除文件夹
	rm -fr  需要移除的文件路目录路径||名称
	
	同时删除多个:
		rm -fr	 文件名称1 文件目录1 文件名称2 文件目录2
		
	删除共性文件:
		rm -f gwl*	*表示任意字符 以gwl开头 后续规则不管
		
		

vim 指令 (vim 是一款文本编辑器)
	vim 文件的路径
	文件可以存在或者不存在
	
	
输出重定向:
	一般命令的输出都会显示在终端中， 有些时候需要将一些命令的执行结果保存到文件中进行后续的分析/统计， 则这时后需要使用到的输出重定向技术
	>	覆盖输出
	>>	追加输出， 不会覆盖原始文件内容 会在原始内容末尾继续添加
	
	ls -l > ls.txt
	
	
cat 指令
	cat 有直接打开一个文件的功能
		cat 文件的路径
	cat 对文件进行合并
		cat 待合并的文件路径1 待合并的文件路径2 待合并的文件路径3 待合并的文件路径4 ...
